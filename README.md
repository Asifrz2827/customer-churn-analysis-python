# 📊 Customer Churn Analysis using Python

## 📌 Project Overview

Customer churn is one of the biggest challenges for telecom companies. This project analyzes customer behavior to identify the factors influencing churn using Python for data analysis and visualization.

The analysis includes data cleaning, exploratory data analysis (EDA), feature relationships, and business insights that can help improve customer retention.

---

## 🎯 Objectives

- Understand customer demographics
- Analyze churn behavior
- Identify factors affecting customer retention
- Generate business insights through visualizations

## 📈 Key Insights

- Customers with month-to-month contracts have the highest churn.
- Electronic check payment users churn more frequently.
- Customers with longer tenure are less likely to churn.
- Higher monthly charges are associated with increased churn.
- Customers without Online Security and Tech Support show higher churn rates.
